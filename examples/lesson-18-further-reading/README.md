# Lesson 18: Further Reading and Resources

This section provides a comprehensive list of reference materials, research papers, and advanced resources for continued eBPF learning.

- [Cilium eBPF documentation](https://docs.cilium.io/en/latest/bpf/)

### Books and Papers
- "BPF Performance Tools" by Brendan Gregg
- Linux kernel source code (kernel/bpf/)
- Various research papers on kernel instrumentation

### Community Resources
- [Eunomia.dev tutorials collection](https://eunomia.dev/)
- [Brendan Gregg's eBPF resources](http://www.brendangregg.com/ebpf.html)
- BPF community mailing lists

- Performance optimization techniques
